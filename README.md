# Abanoub-Kamhy-Shehata
Project Overview

This is a personal portfolio website for Abanoub Kamhy, a Civil Engineer.
The website showcases professional information, technical and soft skills, courses & trainings, work experience, and provides an interactive, modern UI with animations and dark/light mode support.

The project is designed to be responsive, interactive, and visually engaging, providing a professional online presence.

Features
Hero Section

Profile image with zoom-in and hover animation

Typing effect for name and role

Hire Me button linking to contact page

Parallax background slider with multiple images

Floating particles animation for a modern effect

About Me

Interactive About Me box

Describes professional background, skills, and strengths

Styled with backdrop blur for a modern card effect

Skills Sections

Technical Skills: AutoCAD, Revit Structure (BIM), ETABS/SAFE/SAP2000, BOQ & Cost Estimation, Site Supervision & Execution

Soft Skills: Communication, Teamwork, Problem Solving, Time Management, Critical Thinking

Smooth scroll animations as the user navigates the page

Courses & Trainings

List of professional trainings with dates displayed on the right

Structured for clarity and readability

Includes both online and offline courses covering structural design, shop drawings, Revit BIM, and cost management

Experience Page

Separate page for Professional Experience

Includes detailed descriptions, responsibilities, and skills gained

Each card has hover animation and scroll reveal

Dark mode and floating particles consistent with main page

Additional Features

Dark/Light Mode toggle

Scroll progress bar at the top

Fully responsive layout for desktops, tablets, and mobile devices

Smooth transitions and animations for modern UI feel

Technologies Used

HTML5

CSS3 (Custom properties / variables, animations, transitions)

JavaScript (DOM manipulation, scroll animations, background slider, typing effect)

Responsive design (media queries)

Optional: GitHub Pages for hosting
